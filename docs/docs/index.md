# Bem-vindo ao Projeto Databricks 



Este site apresenta a documentação completa das operações realizadas com Delta Lake no ambiente Databricks, com o objetivo de criar e popular as camadas Landing,Bronze, Silver e Gold por meio de um pipeline de dados estruturado.

O projeto utiliza um dataset com informações de Airbnb em oito cidades europeias, servindo como base para as transformações e análises ao longo do processo.

Toda a documentação foi organizada para facilitar o entendimento, o estudo e a reprodução do ambiente, oferecendo uma visão clara das etapas de ingestão, processamento e consumo dos dados.
